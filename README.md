# Tesla-Market-and-Trend-Analysis-Dashboard


## Project Overview

The Tesla Models Dashboard: Innovation in Motion is a data analytics project designed to provide comprehensive insights into Tesla’s performance across production, deliveries, sustainability impact, and infrastructure growth over time.

This interactive dashboard consolidates multi-dimensional data (2015–2025) to help users analyze trends, compare regions, and evaluate different Tesla models efficiently.


## Objectives

The main goal of this project is to:

* Analyze Tesla’s production and delivery trends
* Evaluate environmental impact (CO₂ savings)
* Track charging infrastructure growth
* Compare vehicle specifications and pricing
* Enable region-wise and model-wise filtering for deeper insights

## Dashboard Features

The dashboard was designed using multiple tools (Excel, Power BI, and Tableau) with interactive features to enhance data exploration and user experience.

* **Interactive Filters (Excel & Tableau):**
    * Used *Region* and *Model* slicers in Excel for dynamic data filtering
    * Applied *Region* and *Model* filters in Tableau to enable user-driven analysis
    * These filters allow users to easily explore data across different segments

* **Dynamic & Real-Time Interaction:**

Enhances user engagement and navigation


 ## Key Insights

**Production vs Deliveries Trend:**
Production units (28.13M) are consistently higher than deliveries (26.19M), indicating effective inventory planning and readiness for future demand.

**Balanced CO₂ Savings Across Regions:**
CO₂ savings are almost evenly distributed across Asia, Europe, Middle East, and North America, showing global adoption of electric vehicles.

**Strong Sustainability Impact:**
Total CO₂ savings of 1.96M tons highlights Tesla’s significant contribution toward reducing carbon emissions and promoting clean energy.

**Steady Growth in Charging Infrastructure:**
Charging stations show consistent growth over the years, reflecting Tesla’s focus on supporting EV ecosystem expansion.

**Stable Production Trends:**
Yearly production units remain relatively stable with slight growth, indicating efficient manufacturing operations.

**Model Performance Comparison:**
Tesla models have similar battery capacity and range, but pricing varies slightly, showing strategic product positioning.

**Minor Dip Around 2020:**
A slight decline in production and deliveries is visible, possibly due to global disruptions (e.g., pandemic impact).
